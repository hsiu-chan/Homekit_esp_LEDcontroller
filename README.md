# Homekit_esp_LEDcontroller
Control four LEDs' Brightness by pwm.

# Wifi setting
1. Connect to wifi: ESP8266-LEDctrl-Setup
2. Go to 192.168.4.1
3. Setting your 2.4G wifi

# Homekit password

`111-11-111`

# Hardware 
- Wemos D1 mini
- High Power MOSFET*4
